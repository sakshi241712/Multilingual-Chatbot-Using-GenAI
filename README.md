# Multilingual-Chatbot-Using-GenAI
Chatling-based multilingual chatbot supporting Indian languages

## Live Website
https://sakshi241712.github.io/Multilingual-Chatbot-Using-GenAI/

## Project Overview
This project is a multilingual chatbot-powered website that supports Indian regional languages. The chatbot is built using Generative AI and integrated into the website using Chatling.

## Features
- Multilingual chatbot (Hindi, Marathi, Tamil, Telugu, English)
- AI-powered responses using LLM
- Website hosted using GitHub Pages
- Chatbot embedded using JavaScript

## Technologies Used
- HTML, CSS, JavaScript
- Chatling
- Generative AI (LLM)
- GitHub Pages


## Deployment
The website is deployed using GitHub Pages, and the chatbot is integrated using Chatling's embed script.
